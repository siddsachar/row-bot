## What Is Saved

Configuration is saved under the active Row-Bot data directory.

## Local And External Behavior

The documented UI opens locally. Side effects are disabled during docs capture.

External services are contacted only when credentials/configuration are added and the user starts or invokes the integration.

## Credentials And Safety

Public demo screenshots use example domains, disconnected states, or masked controls.

Actions that can change files, call external systems, start servers, or use MCP tools follow the active approval policy. When a surface is disconnected or empty, Row-Bot keeps the controls visible so you can see what still needs setup.
