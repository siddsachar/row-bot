# Getting Started

Use this section when you are opening Row-Bot for the first time.

1. Install Row-Bot for your operating system.
2. Choose a model path: local, provider, subscription, or custom endpoint.
3. Complete the setup wizard.
4. Start a chat and ask Row-Bot to do a real task.

The detailed guides in this section will become screenshot-backed as the docs
automation pipeline comes online.

## Common First Tasks

- Ask Row-Bot to remember a preference.
- Search the web or read a URL.
- Upload a document and ask questions about it.
- Create a workflow.
- Create a deck or page in Designer Studio.
- Connect a repository in Developer Studio.
- Install a public skill.
