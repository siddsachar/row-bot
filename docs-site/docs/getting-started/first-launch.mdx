# First Launch Setup

On first launch, Row-Bot opens a setup wizard. The wizard helps you choose how
the assistant should run models and which setup areas you want to finish next.

## Model Paths

Row-Bot supports several model paths:

- local models through Ollama;
- hosted provider API keys;
- subscription-backed providers such as ChatGPT / Codex and Claude
  Subscription;
- custom OpenAI-compatible endpoints such as LM Studio, vLLM, llama.cpp,
  LocalAI, LiteLLM, and SGLang.

## Setup Center

After model setup, the setup center can guide you through documents, workflows,
Designer Studio, Developer Studio, channels, accounts, tools, plugins, MCP, and
voice.

The public docs screenshot pipeline will capture this flow from a deterministic
demo app state so users can preview it before installing.
