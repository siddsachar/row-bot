# Developer Studio

Developer Studio connects Row-Bot to a local Git workspace. It can inspect
files, manage code threads, track todos, show diffs, run tests, prepare commits,
push branches, and prepare pull requests. It can also create Custom Tools from
reviewed repositories or folders.

<Screenshot

  alt="Row-Bot Developer Studio home showing demo workspaces and Custom Tools."
/>

## Start Safely

Open or create a workspace, review the branch, and choose an approval mode.
Use ask/approval mode for unfamiliar repositories, destructive shell commands,
or broad file edits. Keep credentials and private files out of prompts unless
you intentionally need them for the task.

  id="developer-workspace"

## Review Before Commit

Before committing, inspect changed files, run the relevant tests, read the diff,
and confirm generated files are intentional. Custom Tools should be reviewed
like code: check permissions, inputs, filesystem access, network behavior, and
failure modes before promoting them into the normal tool surface.
