# Channels And Voice

Channels let Row-Bot communicate through external messaging systems such as
Telegram, WhatsApp, Discord, Slack, and SMS. Voice features include dictation,
read-aloud, realtime talk, transcription, local TTS, and provider-backed voice
sessions.

Automated docs will include:

- channel setup cards;
- health and readiness states;
- delivery defaults;
- voice model settings;
- realtime talk configuration;
- diagnostics.
