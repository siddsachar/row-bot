# Skills, Plugins, And MCP

Row-Bot has several extension layers:

- Smart Skills provide reusable instructions and workflow patterns.
- Skills Hub can browse, preview, scan, and install public skills.
- Plugins add extension bundles and settings.
- Custom Tools turn reviewed repos or folders into reusable tools.
- MCP connects external Model Context Protocol servers.

## Skills

Use bundled skills first when they fit the task. For public skills, preview the
source and scan the instructions before installing. Pin only the skills you want
Row-Bot to consider frequently.

<Screenshot
  id="settings-skills"

## Plugins And Custom Tools

Plugins and Custom Tools can add capabilities beyond the built-in tool set.
Review the manifest, settings, tools, and secrets before enabling a plugin.

## MCP

MCP servers can expose external tools. Prefer recommended servers when possible,
review native capability overlap, and keep destructive tools behind approvals.
