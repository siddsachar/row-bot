# Designer Studio

Designer Studio creates and edits visual work such as decks, documents, landing
pages, app mockups, and storyboards. It supports templates, live preview, brand
controls, critique and repair, image/video generation, charts, Mermaid, Plotly,
export, and shareable HTML.

## Start A Project

Open Designer from Home, choose a project type, and start from a template or
brief. Keep the first brief concrete: audience, format, sections, brand
constraints, and export target.

<Screenshot
  id="designer-new-project"
  alt="Row-Bot Designer Studio new project dialog with template choices."
/>

## Edit And Review

The editor combines a page navigator, live preview, and assistant panel. Ask
for targeted changes such as “tighten the CTA section” or “make the timeline
scannable.” Use review/critique before export when factual or brand accuracy
matters.

<Screenshot
  id="designer-editor"
  alt="Row-Bot Designer Studio editor with page navigator, preview, and assistant panel."
/>

Exports can create files or shareable HTML. Check generated media and any
externally sourced content before publishing.
