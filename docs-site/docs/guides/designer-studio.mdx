# Designer Studio

Designer Studio creates and edits visual work such as decks, documents, landing
pages, app mockups, and storyboards. It supports templates, live preview,
brand controls, critique and repair, image/video generation, charts, Mermaid,
Plotly, export, and shareable HTML.

Automated screenshots will cover:

- Designer gallery;
- new project dialog;
- template selection;
- editor layout;
- preview and page navigator;
- brand, export, share, review, and command palette dialogs.
