# UI Tour

Row-Bot is organized around a few durable work surfaces:

- Chat for everyday conversation, tools, model switching, files, and approvals.
- Home for Workflows, Designer, Developer, Knowledge, and Activity.
- Settings for providers, models, knowledge, Buddy, voice, accounts, channels,
  utilities, MCP, plugins, and system preferences.
- Designer Studio for visual documents, decks, landing pages, mockups, and
  storyboards.
- Developer Studio for repository work, reviews, diffs, tests, custom tools,
  commits, pushes, and PR prep.
- Knowledge for memory, graph review, document-derived knowledge, and recall
  diagnostics.

The automated screenshot phase will add live screenshots for each of these
surfaces.
