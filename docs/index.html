Privacy by architecture

Thoth's app has no account system, server, or telemetry pipeline. Conversations, memory, Designer projects, documents, and local model runs stay on your machine. Cloud models and online tools are opt-in, and API keys use the OS credential store when available.

What’s new in v3.18

  • External MCP tools. Add stdio, Streamable HTTP, and SSE MCP servers from Settings, test them before use, and expose enabled tools under guarded names.
  • Hermes/OpenClaw migration. Preview and import selected identity, memory, skills, model settings, MCP servers, and explicitly selected API keys with backups and redacted reports.
  • Secure API-key storage. Core and plugin keys now use the OS credential store when available, with metadata-only local files and legacy plaintext migration.
  • Cloud defaults fixed. Saved GPT, Claude, Gemini, Grok, and OpenRouter defaults are preserved even when provider caches are empty or temporarily unavailable.
  • 1850 tests passing. Expanded coverage for MCP, migration, secure key storage, cloud defaults, release metadata, and installer packaging.
